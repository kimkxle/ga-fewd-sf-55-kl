    How do you decide between using aside, article, section, or div elements?
    A:
    section - used for grouping related contents ex) chapters within a book.
    
    div - are similar to sections, but has no relations.
    
    article - are independent, self contained items ex) are comment, new story, blog post, forum post.
    
    aside - content that is seperate from the main content in place. e.g. a side note, side bar, etc...
    
    Nav - site wide navigation, ex) most websites has a main navigation row


    Source1: <a href:"http://stackoverflow.com/questions/4781077/html5-best-practices-section-header-aside-article-elements">html5 best practice for differentiation of elements</a>
    Source2: <a href:"https://www.w3schools.com/tags/tag_article.asp">article html tag</a>

The div elements can be replaced with the new elements: header, nav, section, article, aside, and footer.

    How should the alt attribute for img be used? What are the usability/SEO implications?
    A:
    alt -  describes an attribute or description for the image if not shown. Important because it specifies an alternate text if the image can not be displayed. Additionally, it's a text alternative for search engines. Search enginges / robots can not interpret images,
    ex) <img src="jpeg" alt="display image">

    source: https://www.w3schools.com/tags/att_img_alt.asp
